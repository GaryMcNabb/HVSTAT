Export - The data can be exported as a TSV (tab-separated-values) format text file.

Import - An exported TSV file can be merged into the database. The rows in the file that have older date will be skipped.

Delete - You can delete specific data if you want to clean them.

Note: 'Chance PCT.' values on Drops tab will be incorrect if you import the data of Item Drops after resetting Drops tab.

Delete IndexedDB Database

Delete if there is a problem with the database. The database will be re-created after the next page load. Then you can import the data previously exported.

Monster Database Migration

Migrate to New Database

HV STAT will now use the new IndexedDB database instead of the localStorage database. You can migrate your old database into the new database.

Note: If you have already scanned, note that this operation will overwrite existing data on the new database.

Delete Old Monster Database on localStorage

Currently your old monster database occupies MB on the localStorage. In order to free up space for other HV scripts, delete the old database after migration.
